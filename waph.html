Course Overview

Web Application Programming and Hacking, or WAPH, is a course designed to teach basic web design concepts, common vunerabilities, and how to prevent them through the use of hands-on activities and projects. These activities are as follows:

Labs

Labs are assignments geared toward introducing new concepts in the form of walkthrough activities.

  • Lab 0 - Development Environment Setup
  • Lab 1 - Foundations of the Web
  • Lab 2 - Front-end Web Development
  • Lab 3 - Secure Web Application Development in PHP/MySQL
  • Lab 4 - A Secure Login System with Session Authentication

    • Hackathons

    Hackathons are assignments that involve trying to exploit some vulnerability on a web application designed by the instructor.

  • Hackathon 1 - Cross-site Scripting Attacks and Defenses
  • Hackathon 2 - SQL Injection Attacks
  • Hackathon 3

    • Individual Projects

    Individual Projects are assignments that allow the students more freedom to apply the concepts learned in class, as well as self-taught concepts. One example would be creating a resume website and using open source JS frameworks to implement different features in the webpage.

  • Individual Project 1

    • Team Project

    The Team Project is an opportunity to work in a small group and use agile development techniques to design a web application.

  • Team Project
